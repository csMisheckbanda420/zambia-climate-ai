# Zambia Climate Early Warning System

An AI-powered system for predicting droughts and floods in Zambia using machine learning and real-time data visualization.

## Features

- 🌍 Interactive maps showing flood and drought risks
- 📊 Real-time trend analysis with 7-day and 30-day averages
- 🚨 Automated alert system for high-risk events
- 🔍 City search functionality
- 📈 30-year historical climate data
- 🤖 Machine learning predictions using XGBoost
